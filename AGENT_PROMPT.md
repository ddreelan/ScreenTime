# Agent Prompt for Screen Time Management App

## Overview
Design an iOS/Android app focused on managing screen time, incorporating features that help users balance their device usage with real-life activities. The app should allow for customizability in how screen time is rewarded or penalized, along with verification mechanisms to ensure user activities align with the goals set.

## Key Features
1. **Screen Time Tracking**  
   - Monitor daily and weekly screen time across various apps and categories.
   - Provide insights and analytics to users on their usage patterns.

2. **Configurable Reward/Penalty System**  
   - Users can set their own rewards (e.g., extra screen time, access to premium content) or penalties (e.g., restricted access to certain apps) based on their performance in managing screen time.
   - Implement different reward tiers to encourage users towards better screen time practices.

3. **Activity Verification**  
   - Use geolocation or device sensors to verify user activities outside of screen time (e.g., outdoor activities, exercise) that qualify for rewards.
   - Support integration with fitness apps and trackers for comprehensive activity verification.

4. **Customization and User Profiles**  
   - Allow users to create profiles and tailor their experience based on age, goals, and preferences.
   - Provide templates or examples for suggested settings based on common user objectives.

5. **Notifications and Reminders**  
   - Implement reminders for users to take breaks or engage in real-life activities.
   - Encourage users with motivational messages and positive reinforcement based on their progress.

6. **Community and Sharing**  
   - Create features for users to connect with friends or families to share achievements and challenges.
   - Include options for group challenges or competitions to encourage collective improvement in screen time management.

## Design Considerations
- Ensure usability and accessibility for users of all ages.
- Prioritize a clean, intuitive interface that makes navigation seamless.
- Incorporate gamification elements to enhance user engagement without becoming counterproductive.

## Technical Requirements
- Cross-platform support for iOS and Android.
- Implement robust data privacy and security measures to protect user information.
- Plan for potential integrations with other health or lifestyle applications.

## Conclusion
This app aims to promote a healthy relationship with technology by empowering users to take control of their screen time while providing necessary tools and support along the way.
